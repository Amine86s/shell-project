Linux Multi-Shell Terminal Environment

A GUI-based multi-shell terminal environment developed using C and GTK4. This project focuses on Linux system programming concepts such as process management, IPC, shared memory, semaphores, synchronization, and command execution.

Features
Multiple shell instances
Shared memory communication
Semaphore synchronization
Pipe and redirection support
GTK4-based GUI terminal interface
Command parsing and execution
Concurrent shell management
Technologies
C Programming Language
GTK4
POSIX Shared Memory
POSIX Semaphores
Linux System Calls
Concepts Used
Process Management
Inter-Process Communication (IPC)
Synchronization
Concurrency
MVC Architecture
Run :
make
./terminal_app
